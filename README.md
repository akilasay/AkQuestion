# AkQuestion
This is an application to practice questions
User can create a question and  practise the answering it. 
# Setup the project locally
download the zip file or using git --> git clone https://github.com/akilasay/AkQuestion.git
open using your IDE ( ex - IntelliJ IDE )
add your mysql db username and password to application.properties file 

# Run
In Terminal --> mvn spring-boot:run |  use Maven to run the spring boot application
